atributos: son propiedades o configuraciones que modifican el elemento o etiqueta.

class es un atributo que se le da a uno o mas elementos y se pueden aplicar varios.
id es un atributo que le da una identificacion unica a la etiqueta o elemento.

SELECTOR-PROPIEDAD-VALOR
Selector es la referencia de acuerdo al orden que se le asigne conforme al uso que tenga
Propiedad es la caracteristica que queremos cambiar
Valor dira cualsera el estilo de esa propiedad

Selector universal es *

Selector para una class es .

Selector para una id es #

Selector para una etiqueta es su nombre
